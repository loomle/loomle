# Loomle 0.7.0-rc.1

This is the first release candidate of Loomle 0.7, rebuilt around SAL
(Structured Agent Language) and a standalone MCP Client bundled with the Unreal
Engine plugin.

## Supported environment

- Unreal Engine 5.7
- macOS on Apple Silicon
- Engine-plugin installation

Remove or move any old project-local `Plugins/LoomleBridge` copy before using
0.7. Unreal gives a same-named project plugin precedence over the engine
plugin.

## Highlights

- Readable, copyable SAL queries and patches for complex Unreal objects.
- Asset, Blueprint, Class, Graph, Widget, and StateTree interfaces.
- Blueprint execution-flow, data-flow, schema, diagnostics, and reference
  discovery.
- Editor context and explicit multi-project discovery/binding.
- Shared mutation preflight, dry-run, transaction, undo, compile, and save
  contracts.
- Standalone Client and Bridge protocol versioning with packaged end-to-end
  verification.

## Current limits

- This candidate supports only UE 5.7 on Apple Silicon Macs.
- Windows packaging and verification are not included yet.
- Runtime UI observation, animation/navigation editing, automatic Blueprint
  layout, and cross-Graph execution expansion remain outside this candidate.

## macOS security notice

This release candidate is not yet signed with a Loomle Developer ID certificate
or notarized by Apple. macOS Gatekeeper may therefore require the user to
explicitly approve the downloaded plugin or Client. Stable Loomle releases will
remain blocked until signing and notarization are part of the exact artifact
path tested before publication.
