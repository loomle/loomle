# Blueprint Class

Blueprint class tools handle the class contract of a Blueprint asset.

Use:

- `blueprint.inspect` for asset-level Blueprint summary.
- `blueprint.class.inspect` for parent class and implemented interfaces.
- `blueprint.class.edit` to change the parent class or interfaces.

Class tools do not edit graph nodes, member signatures, or pin defaults.
