# Widget Tools

Widget tools operate on UMG WidgetBlueprint assets.

- `widget.palette`
- `widget.tree.inspect`
- `widget.tree.edit`
- `widget.inspect`
- `widget.compile`

Use `widget.palette` before adding widgets, and `widget.tree.inspect` before
editing the WidgetTree hierarchy.
