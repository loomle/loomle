# LOOMLE Client

This directory contains the installed project-local `loomle` client binary.

Installed runtime:

```text
Loomle/client/loomle(.exe)
```

Do not treat this file as the main usage guide.

Read `../README.md` first for:
- the full command overview
- when to use `call` versus `session`
- install and upgrade guidance
- when to open `workflows/` or `examples/`

This directory exists so the project-local client has a stable home inside the workspace.
