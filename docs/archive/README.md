# Archive

This directory contains historical repository material that is no longer part
of the current product contract.

`workspace/` preserves the legacy Loomle workspace guides, catalogs, examples,
prototype skills, and generation helpers. Files inside it may intentionally
refer to old paths such as `workspace/Loomle` and old tool names such as
`graph.mutate`.

`legacy/0.6/` preserves the retired 0.6 distribution design and direct-tool
contracts. Names such as `blueprint_graph_edit`, `addFromPalette`,
`blueprint_member_edit`, `widget_tree_edit`, and `schema_inspect` are historical
evidence, not current 0.7 interfaces.
